Module: lib/el

Methods

(inner) el(type, options) → {object}

Convenience method that allows you to create a DOM element and assign an id, classes (space separated string) and styles to it.
Parameters:
Name Type Description
type string a string that represents the type of DOM element you want to create
options object An object that contains css styles
Properties
Name Type Description
classes string A space seperated list of classes
id string an identifier
styles object an object that contains all the styles you want to apply to the DOM element (and override the class-styles with)
Properties
Name Type Description
modalContainer object styles for the container
modalOverlay object styles for the overlay
modalClose object styles for the close cross/button
modalContent object styles for the content
modalFooter object styles for the footer
modalBody object styles for the body
modalHeader object styles for the header

Example
import el from './lib/el'

const divEl = el('div', {
  classes: 'myCls1 myCls2',
  id: 'myDiv',
  styles: {
    modalOverlay: { backgroundColor: 'red' }
  }
})
