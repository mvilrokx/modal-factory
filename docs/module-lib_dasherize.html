Module: lib/dasherize

(require("lib/dasherize"))(str) → {string}

Converts the camelCased/CamelCased string that is passed in to a dash'ed string. This is handy to convert JavaScript object keys and we use it to convert object styles to css styles, e.g. borderWidth becomes 'border-width'
Parameters:
Name Type Description
str string camelCased string you would like to convert

Example
import dasherize from './lib/dasherize'

console.log(dasherize('camelCase')) //'camel-case'
