Module: lib/styleObjectString

(require("lib/styleObjectString"))(style) → {string}

Converts an object that contains CSS styles into a string. Returns a string of semi-colon-seperated css that can be easily assigned to the cssText proporty of a DOM element
Parameters:
Name Type Description
style object An object that contains css styles
Author:
  • Mark Vilrokx
Source:
  • styleObjectString.js, line 28
Returns:
a string that contains a semi-colon-seperated list of css styles
Type
string
Example
import style2str from './lib/styleObjectString'

console.log(style2str({
  backgroundColor: '#dddddd',
  borderTopLeftRadius: '3px',
}) // 'background-color: #dddddd; border-top-left-radius: 3px; '
